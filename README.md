# オフライン漢字辞書

CSVを読み込んで使う、ローカル動作の漢字辞書ツール。自分用。

## 使い方

1. `index.html` をブラウザで開く
2. CSVファイルを選択する
3. 部首ボタンで絞り込みができる

## CSVの形式

1行目はヘッダー行（読み飛ばされる）
```
単語,読み,部首,説明,画像
木,き,木,木を表す漢字。,pic/ki.png
```
- 画像は `index.html` と同じフォルダを基準にしたパスで指定
- 画像なしの場合は5列目を空欄にするか省略でもOK

## 動作環境

ブラウザのみ。インターネット接続不要。

自由に改変してください。
